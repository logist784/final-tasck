Git

  • git status: Отображает текущее состояние репозитория, включая измененные, добавленные и не отслеживаемые файлы.
  • git clean: Удаляет неотслеживаемые файлы.
  • git clone : Создает локальную копию удаленного репозитория.
  • git branch: Перечисляет, создает или удаляет ветки.
  • git log: Отображает историю коммитов/
